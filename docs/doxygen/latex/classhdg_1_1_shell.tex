\doxylink{classhdg_1_1_shell}{Shell} 的运行分为 release 模式和 debug 模式。 其中 debug 模式会在每次输入命令后打印出 tokens 列表与语法树，而 release 模式则会直接输出结果。 默认情况下，以 release 模式启动 Shell。~\newline
~\newline
 除此之外，如果输入文件路径，那么 \doxylink{classhdg_1_1_shell}{Shell} 会直接解析文件里的代码；否则直接进入到命令行。 读取传入的参数，并且解析传入的参数。
