\doxysubsection{Detailed Description}
词法分析器 

所有传入 \doxylink{classhdg_1_1_interpreter}{Interpreter} 的源代码首先都要传入到该类中进行解释，将文本数据加工为 toknes 列表。 并且记录相对行号列号。 \begin{DoxyAttention}{Attention}
词法分析器不存储源代码具体是什么，只记录已经读取过多少行代码。所以，每次调用 run 函数，都要重置 m\+\_\+pos 
\end{DoxyAttention}
